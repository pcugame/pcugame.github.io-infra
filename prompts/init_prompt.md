1. 2021, 2022, 2023...과 같은 여러 해의 정보를 보여줄 수 있어야 합니다. 해당 해의 페이지에 들어가면, 각 게임이 하나의 그리드를 차지하고 제목 / 이름 / 학번 / 자세히보기 정도의 버튼을 가지고 있어야 합니다.
2. Github Pages에 정적으로 빌드 후 업로드하여 호스팅 합니다.
3. 서버에서 데이터를 받아서 출력합니다.
4. 서버는 Synology NAS로, DBMS를 Docker 하에서 동작시키고 웹페이지는 해당 DBMS에게 정보를 요청해야 합니다.
5. ORM과 같은 최신 기술들은 필요할 경우(관리, 유지보수가 용이하거나, 개발 편의성을 늘린다면) 사용해도 좋습니다.
6. 시각 디자인이 아니라 동작과 구조를 구현해야 합니다.
7. 로그인 기능이 구현되어야 합니다. Google OAuth 방식이어야 하고, 학교 도메인 메일주소로만 로그인이 가능해야 합니다.
8. 동영상은 Youtube에서만 호스팅 될 것이고, 사진과 게임파일은 무조건 Synology NAS에서 안전한 방법으로 호스팅되어야 합니다.
